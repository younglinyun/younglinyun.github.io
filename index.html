PGI Fortran + Intel MKL 安装使用

发表于 2019-05-16 | 更新于 2019-05-17 | 分类于 笔记

安装 PGI Community Edition

前往官网的下载页面，下载免费社区版，选择Windows系统即可。首页底部的Documentation-Installation Guide里面有对Linux、MacOs和Windows操作系统下安装过程详细的说明。其中Windows部分在第6节，这里给出简单的总结：

安装步骤

正式安装之前，应确保PGI软件已下载，而且其他必需的非PGI软件已经安装完毕，具体而言就是如下3个：

  1. Microsoft Update for Universal C Runtime，通常而言，Windows的自动更新已经确保它已经安装上。

  2. Microsoft Windows SDK

  3. Microsoft Visual Studio 2017

最后两项安装 Visual Studio 2017 社区版就行，工作负载选择使用 c++ 的桌面开发，可以看到右侧安装详细信息里默认有 Windows 10 SDK 选项，其他选择默认即可，大概需要空间 6.6G。

最后以管理员权限运行 PGI 安装包即可，最后把 bin 文件夹需要添加至环境变量。
